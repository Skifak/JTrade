# Иконки приложения

`tauri build` требует наличия иконок: `32x32.png`, `128x128.png`,
`128x128@2x.png`, `icon.icns`, `icon.ico`.

`tauri dev` работает и без них — можно сразу запускать `npm run tauri:dev`.

Для билда — сгенерируй иконки **один раз** из любой квадратной PNG-картинки
(желательно 1024×1024) встроенной командой Tauri:

```powershell
npm run tauri:icon ./path/to/source.png
```

Команда положит сюда все нужные размеры (PNG / ICO / ICNS / разные плотности
для разных ОС).

После генерации файлы должны коммититься в репозиторий, чтобы билд можно было
собрать на чистой машине без ручной подготовки.
